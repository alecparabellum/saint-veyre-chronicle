# Visual Continuity Checklist

Use before approving any repeated depiction of an established character.

## Identity

- [ ] Facial structure matches canonical references.
- [ ] Apparent age remains correct.
- [ ] Skin tone and undertones remain consistent.
- [ ] Eye shape and colour remain consistent.
- [ ] Distinguishing marks are present and correctly placed.

## Body

- [ ] Height impression remains plausible.
- [ ] Body type and proportions remain consistent.
- [ ] Head-to-body proportions are correct.
- [ ] Hands and limbs are anatomically coherent.

## Hair

- [ ] Texture is correct.
- [ ] Length is correct.
- [ ] Established silhouette or hairstyle is preserved unless explicitly changed.
- [ ] Hairline and volume remain plausible.

## Styling

- [ ] Signature fashion identity is preserved.
- [ ] Defining accessories are present when appropriate.
- [ ] New clothing feels like a believable extension of the character.
- [ ] Grooming and makeup remain within the approved range.

## Visual Grammar

- [ ] Palette belongs to the character.
- [ ] Composition reflects the character's profile.
- [ ] Posture and gesture communicate the correct temperament.
- [ ] Background belongs to the character's world.
- [ ] The shadow-self contradiction is subtle and relevant.

## Canon Control

- [ ] No unapproved trait has silently replaced an established one.
- [ ] Temporary scene-specific changes are not recorded as permanent canon.
- [ ] Any approved new canonical trait is added to the visual profile.
